### Lista de exercícios sobre MPI

A lista com os enunciados dos exercícios se encontra na pasta *listas*. Os arquivos *exercicio1.c*, *exercicio2.c*, *exercicio3.c*, *exercicio4.c* e *exercicio5.c* são as resoluções do arquivo *exercicios.pdf*. E o arquivo *trapezio.c* é a resolução do primeiro exerício do arquivo *trapezio.pdf*.

#### Execução

Basta executar o comando de compilação `mpicc nomeDoArquivo.c -o nomeDoArquivo` e rodar o comando de execução `mpirun -np n nomeDoArquivo`, onde n = quantidade de processos.
